# Keycube Heatmap 3D

Interactive 3D visualization of [Keycube](https://keycube.io) user study data.

**🔗 [Live Demo](https://keycube.github.io/heatmap3d)**

## Modes

- **Preference** — Key preferences per participant (1–10 scale)
- **Reachability** — Finger-to-key accessibility heatmap

## Run locally

```bash
bundle install
bundle exec jekyll serve
```
